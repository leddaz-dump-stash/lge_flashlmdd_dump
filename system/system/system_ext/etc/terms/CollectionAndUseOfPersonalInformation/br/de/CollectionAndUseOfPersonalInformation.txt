2.	 Übersicht über FOTA-Softwareaktualisierungen
Von Zeit zu Zeit kann es erforderlich sein, die LGE-Firmware auf Ihrem Mobiltelefon (die „Software“) zu aktualisieren oder es müssen Ergänzungen vorgenommen werden, die von LGE bereitgestellt werden, oder für die LGE möglicherweise Software-Zusatzkomponenten bereitstellt. Mit Softwareaktualisierungen bekommen sie die Möglichkeit, neue Eigenschaften oder Funktionen hinzuzufügen und in bestimmten Fällen auch vorhandene Eigenschaften oder Funktionen zu entfernen. Nach dem Fertigstellen der Softwareaktualisierung können Sie möglicherweise nicht mehr auf die zuvor installierte Version der Software zurückgreifen. 

Da die Softwareaktualisierung über einen FOTA-Vorgang durchgeführt wird, können, abhängig von Ihrem Netzbetreiber und Ihrem Tarif, zusätzliche Kosten anfallen, wann immer Sie neue Software über ein Mobilfunknetz herunterladen. Falls Sie eine Verwendung von Mobilfunkdaten für Softwareaktualisierungen vermeiden möchten, wählen Sie unter Einstellungen die Option „Nur Wi-Fi“ aus. 

a.	 Ihre Optionen für die Verwaltung zukünftiger Software-Updates
Wenn Sie dieser FOTA-Datenschutzrichtlinie zustimmen, wird Ihr Mobiltelefon von Zeit zu Zeit nach neuer oder aktualisierter Software suchen und neue oder aktualisierte Software automatisch installieren, sobald diese verfügbar ist ("Automatische Überprüfung"). Sie werden die Möglichkeit haben, die Software-Aktualisierung zu akzeptieren oder abzulehnen, bevor Sie die Software anwenden. Ihr Mobiltelefon wendet die installierte Software ebenfalls automatisch um Mitternacht an (Funktion "Automatische Updates"). Sie können diese Funktionen "Automatisch prüfen" und "Automatische Updates" unter [Einstellungen - System - Update Center - Software-Aktualisierung - Overflow-Menüs - Einstellungen] ablehnen.  Wenn Sie "Automatisch prüfen" ablehnen, lehnen Sie die Durchführung zukünftiger Software-Updates effektiv ab.
